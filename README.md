# 📊 Pandas – Practical Foundations for Data Analysis

This repository contains my structured practice and hands-on learning of **Pandas**, a powerful Python library used for data manipulation and analysis.

The goal of this repository is to build strong foundational skills required for **Data Analyst / Business Analyst / BI roles**.

---

## 📂 Repository Structure

The notebooks in this repository cover the core building blocks of Pandas:

### 1️⃣ Series

Understanding:

* Creating Series
* Indexing
* Basic operations

### 2️⃣ DataFrames

* Creating DataFrames
* Selecting rows & columns
* Filtering data
* Basic exploration (`head()`, `info()`, `describe()`)

### 3️⃣ Missing Data Handling

* Detecting null values
* `isna()`, `fillna()`, `dropna()`
* Handling incomplete datasets

### 4️⃣ Merging, Joining & Concatenation

* Combining multiple DataFrames
* `merge()`
* `join()`
* `concat()`
* Different types of joins (inner, left, outer)

### 5️⃣ GroupBy & Aggregation

* `groupby()` operations
* Aggregation functions (`sum`, `mean`, `count`)
* Multi-level grouping

### 6️⃣ Pivot Tables

* Reshaping data
* Creating pivot tables
* Summarizing data across categories

### 7️⃣ Data Operations

* Column transformations
* Feature creation
* Boolean masking
* Applying functions
